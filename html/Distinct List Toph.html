Distinct List

Limits 5s, 256 MB

Byteland has a network of nnn computers that forms a tree structure. A tree is a connected graph that has no cycles.

Each computer uuu has a special database and special software. The database has initially a list LuL_uLu​. The list LuL_uLu​ has only one element AuA_uAu​ initially. The software is programmed to do a special task. When a list BBB is given to the computer uuu’s special software, it starts to iterate through the list BBB. For each element BiB_iBi​ of the list BBB, the software does the following:

  • If the element BiB_iBi​ already exists in the list LuL_uLu​ the software does nothing. Otherwise, it appends BiB_iBi​ to the end of the list LuL_uLu​.

After completing the iteration, the software then sorts the list LuL_uLu​ in ascending order. For example, if a list LuL_uLu​ is [13][13][13] initially and we give the lists [13,75,26],[0,7],[0,7,13,25][13, 75, 26], [0, 7], [0, 7, 13, 25][13,75,26],[0,7],[0,7,13,25] to the software of computer uuu one by one, then eventually the list LuL_uLu​ will be [0,7,13,25,26,75][0, 7, 13, 25, 26, 75][0,7,13,25,26,75].

To ensure the security of the network, each computer sends a copy of all the lists in its database to its parent computer (if exists) only. When a computer uuu gets the lists from a child computer, the computer uuu adds the lists to its database and also gives the lists to its special software to update the list LuL_uLu​. Only after receiving all the lists from all the children's computers and updating the list LuL_uLu​ accordingly, a computer uuu sends a copy of all the lists in its database to its parent computer (if exists).

For each computer uuu, you have to tell how many distinct lists are present in its database after completing all the above operations. Two lists are different if either they have different sizes or one list has at least one element that is not present in the other list.

Input

Input starts with an integer T(1≤T≤105)T(1\leq T\leq 10^5)T(1≤T≤105), the number of test cases.

In each test case, the first line contains an integer n(1≤n≤105)n(1\leq n\leq 10^5)n(1≤n≤105), the number of computers.
The next line contains nnn space-separated integers Ai(0≤Ai≤109)A_i(0\leq A_i\leq 10^9)Ai​(0≤Ai​≤109), the single initial element of the list LiL_iLi​ of computer iii.
The last line of each test case contains nnn space-separated integers Pi(1≤Pi≤n)P_i(1\leq P_i\leq n)Pi​(1≤Pi​≤n), the parent computer of computer iii. If computer iii is the root of the tree, PiP_iPi​ will be −1-1−1.

It is guaranteed that the computer network will form a tree structure and the sum of nnn over all test cases will not be greater than 10610^6106. It is also guaranteed that if n>104n>10^4n>104, then T=1T=1T=1.

Output

In each test case, print nnn space-separated integers in a line. The ithi^{th}ith integer represents the number of distinct lists present in the database of the computer iii.
